# 浮遊する画像アニメーション

このプロジェクトは、React + TypeScript + Vite を使用して作成された、浮遊する画像アニメーションのデモアプリケーションです。

https://tag1-mu.vercel.app/


## 機能

- 画像の浮遊アニメーション
- レスポンシブデザイン
- モダンな UI/UX

## 技術スタック

- React
- TypeScript
- Vite
- Tailwind CSS

## 開発環境のセットアップ

1. リポジトリのクローン

```bash
git clone https://github.com/helpmark-pink/tag1.git
```

2. 依存関係のインストール

```bash
npm install
```

3. 開発サーバーの起動

```bash
npm run dev
```
